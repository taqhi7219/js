The difference between undefined and not-defined .

UNDEFINED NOT-DEFIND
1. It works like when we declared a variable in the code but did not assign the value before printing the variable value. 1. It works like when we did not declare the variable and try to call that variable.
2. "undefined" is a value that can be assigned to a variable. 2. "not defined" indicates that a variable does not exist.
3. console.log(a); // undefined 3. var b = 7; // Assign value 7 to b console.log(a); // not defined
